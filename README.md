## `$ whoami`

ML Engineer working at the intersection of research and production. I design and deploy machine learning systems that hold up under real operational constraints — from training loop to serving infrastructure.

My current focus is **human movement analysis** and **AI-assisted coaching**: building perception pipelines that translate biomechanical signals into real-time, interpretable output.

I debug with logs, think in systems, and write code that runs reliably the day after I push it.

Also: the coffee is non-negotiable. ☕
